# rversions â€” past and present R versions

## Installation


```r
devtools::install_github("metacran/rversions")
```

## Usage

### R-release and R-oldrel


```r
library(rversions)
r_release()
```

```
#> $version
#> [1] "3.1.2"
#> 
#> $date
#> [1] "2014-10-31T08:11:32.082768Z"
```

```r
r_oldrel()
```

```
#> $version
#> [1] "3.0.3"
#> 
#> $date
#> [1] "2014-03-06T08:12:33.995105Z"
```
